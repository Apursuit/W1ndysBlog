### 实验一 阶乘

一、实验目的和要求

熟悉一种编程环境及基础编程练习

二、实验内容

三、主要仪器设备

- 计算机
- 编程语言：Python

四、实验方法与步骤

1. 打开编程环境，编写程序
2. 通过输入 3、5、7、 10 等数值验证程序的正确性

五、主要代码

i = input("请输入一个整数：")
i = int(i)


def jiecheng(n):
    if n == 1:
        return 1
    else:
        return n * jiecheng(n - 1)
