## C 语言代码

### 整理前

```c
int fun(int x, int y)

{

    int local_4;
    int local_14 = x;
    int local_18 = y;
    int eax = local_14;
    if (eax <= local_18)
    {
        eax = local_18;
        local_4 = eax;
    }
    else
    {
        eax = local_14;
        local_4 = eax;
    }
    return local_4;
}

int main()
{
    printf("fun(10,20)=%d\n", fun(10, 20));
    return 0;
}
```

### 整理后

```c
#include <stdio.h>

int fun(int x, int y)
{
    if (x <= y)
    {
        return x;
    }
    else
    {
        return y;
    }

}

int main()
{
    printf("fun(10,20)=%d\n", fun(10, 20));
    return 0;
}
```
